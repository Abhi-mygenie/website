# MyGenie POS — Demo Video Pre-Scripts
> Written by: Script + Product Expert
> Based on: website content, real customer proof, product modules
> Format: Each script = Hook → Problem → Solution (screen walkthrough) → Proof moment → CTA
> Target viewer: Restaurant/café/hotel owner, decision-maker, non-technical
> Tone: Confident, fast-paced, no fluff. Show — don't tell.

---

## HOW TO USE THIS DOCUMENT

- **GIFs (Pricing page):** 8–15 sec looping. No audio. Text overlays only. Show ONE killer moment.
- **Product Videos:** 60–90 sec. Voiceover or captions. Show real app screens.
- **AI Videos:** 30–45 sec. Screen recording + caption overlays. One feature = one moment.
- **Screen to record:** MyGenie app on a phone + tablet (dual device shots look great).
- **Golden rule:** The first 3 seconds must show the PAIN. The next 5 seconds show the SOLUTION. The last seconds show the RESULT.
